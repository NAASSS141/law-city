# 法域之城：雨夜证词 V3

这是一个可直接部署到 GitHub Pages 的静态网页游戏。无需后端，无需 npm，无需构建。

## V3 新增内容

- 新增主菜单的案件选择
- 新增章节导航
- 新增城市地图
- 新增第二个案件：沉默账本
- 新增医院、旧码头、第二案法庭
- 证据扩展到 18 条
- 组合推理扩展到 5 条
- 第一案真相结局可解锁第二案
- 新增最终结局评分：综合评级、真相值、正义值、证据数
- 增加更多角色：护士李娜、私家侦探莫野
- 保持纯静态结构，可直接上传 GitHub Pages

## 文件结构

- `index.html`：游戏入口
- `style.css`：视觉和响应式布局
- `game.js`：剧情、证据、分支、组合推理、章节和结局逻辑
- `assets/`：SVG 场景和角色素材

## 本地预览

直接双击 `index.html` 即可运行。

也可以用本地服务器：

```bash
python -m http.server 8000
```

打开：

```text
http://localhost:8000
```

## 部署到 GitHub Pages

1. 新建 GitHub 仓库，例如 `law-city-game`
2. 解压本 zip
3. 上传解压后的全部文件到仓库根目录
4. 打开 Settings → Pages
5. Source 选择 `Deploy from a branch`
6. Branch 选择 `main`，目录选择 `/root`
7. 保存后等待 GitHub 生成网址

## 玩家通关提示

第一案好结局需要完成以下组合推理：

- 重构案发时间线
- 证明现场被二次处理
- 锁定店长乔衡的动机

第二案好结局需要完成以下组合推理：

- 医院线索指向账本
- 旧码头不是交易终点
