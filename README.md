# 法域之城：雨夜证词 V9

这是一个可直接部署到 GitHub Pages 的静态网页游戏。无需后端，无需 npm，无需构建。

- 新增手动存档系统（3 个槽位）
- 支持手动读档 / 覆盖存档 / 删除槽位
- 新增案件时间线面板
- 继续保留 V8 的三案完整结构
- 保留 V7 的动态配乐、环境音混音与转场演出
- 保留 V6 的 AI 封面、AI 场景插画、AI 角色立绘与角色档案

- 3 个案件：雨夜证词、沉默账本、灰塔匿名函

- 24 条证据
- 7 条组合推理
- 多结局与最终评分
- 手动存档系统 + 自动存档
- 角色档案、视觉图鉴、证据图鉴、成就系统、结局收藏
- 动态配乐、环境音、交互音效
- 本地自动存档
- 纯静态文件，可直接上传 GitHub Pages

## 本地预览

直接双击 `index.html` 即可运行。

也可以使用本地服务器：

```bash
python -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

## 部署到 GitHub Pages

1. 新建 GitHub 仓库，例如 `law-city-game`
2. 解压本 zip
3. 上传解压后的全部文件到仓库根目录
4. 打开 Settings -> Pages
5. Source 选择 `Deploy from a branch`
6. Branch 选择 `main`，目录选择 `/root`
7. 保存后等待 GitHub 自动生成网址
